You can join with LEFT joins all the distinct names of the 3 tables with each of the tables:
select t.name,
coalesce(v.management, o.management) management,
coalesce(v.dns, o.dns) dns,
c.env,
c.type
from (
select name from cmdb union
select name from vmv union
select name from vmo
) t
left join cmdb c on c.name = t.name
left join vmv v on v.name = t.name
left join vmo o on o.name = t.name
See the demo.
Or:
select t.name,
n.management,
n.dns,
c.env,
c.type
from (
select name from cmdb union
select name from vmv union
select name from vmo
) t
left join cmdb c on c.name = t.name
left join (
select name, dns, management from vmv
union all
select name, dns, management from vmo
) n on n.name = t.name
See the demo.
Results:
| name | management | dns | env | type || ---- | ---------- | ---------- | ---- | -------- || SP | | | Dev | Physical || VMO1 | Man2 | VMO1.local | Dev | Virtual || VMO2 | Man2 | VMO2.local | | || VMV1 | Man1 | VMV1.local | Prod | Virtual || VMV2 | Man1 | VMV2.local | | |
