Used to manage and monitor RAID controllers (Disques, controleurs etc.)
    
Remplace HPACUCLI :  HPACUCLI - HP Array Configuration Utility CLI
    
# hpssacli help
    
Commande HP sur ESXi
    
Note : Je pense que hpssacli a été remplacé par ssacli dans les nouvelles versions.

[root@xxxx~]# hpssacli
