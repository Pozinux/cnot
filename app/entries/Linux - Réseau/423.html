Certificat 
   
J'ai déjà un certificat letsencrypt sur mon domaine "tpoz.fr"
     
Copy the certificates information into the cockpit certificate folder using the following commands (créer le fichier s'il n'existe pas) :
     
Note : Il faudra refaire cette manip à chaque fois que le certificat est renouvelé (tous les 3 mois)
     
cat /etc/letsencrypt/live/tpoz.fr/fullchain.pem > /etc/cockpit/ws-certs.d/1-my-cert.cert     
cat /etc/letsencrypt/live/tpoz.fr/privkey.pem >> /etc/cockpit/ws-certs.d/1-my-cert.cert
   
  
Changement de port
  
https://access.redhat.com/solutions/2761591
   
mkdir /etc/systemd/system/cockpit.socket.d/  
   
# cat /etc/systemd/system/cockpit.socket.d/listen.conf  
[Socket]  
ListenStream=  
ListenStream=7070  
   
# systemctl daemon-reload  
# systemctl restart cockpit.service cockpit.socket  
   
# cp /usr/lib/firewalld/services/cockpit.xml /etc/firewalld/services/cockpit.xml  
   
# cat /etc/firewalld/services/cockpit.xml  
<?xml version="1.0" encoding="utf-8"?>  
<service>  
  <short>Cockpit</short>  
  <description>Cockpit lets you access and configure your server remotely.</description>  
  <port protocol="tcp" port="7070"/>  
</service>  
   
systemctl restart firewalld.service
