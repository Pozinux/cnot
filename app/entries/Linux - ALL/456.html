Lister pour voir les types de fichiers (met un caractère spécial à la fin du nom) :
   
/ : répertoire
* : fichier exécutable
= : socket
| : fichier de type FIFO
@ : lien symbolique
   
avec l'option en ligne en plus :
   
[root@centos1 etc]# ls -Fm
abrt/, acpi/, adjtime, aliases, aliases.db, alsa/, alternatives/, anacrontab, asound.conf, at.deny, audisp/, audit/, autofs.conf, autofs_ldap_auth.conf, auto.master,
auto.misc, auto.net*, auto.smb*, b

TREE 
   
Par défaut, tree affiche les répertoires et n'affiche pas les fichiers cachés :
   
[root@centos1 ~]# tree /home
/home
├── nfs
└── tim
   
Afficher seulement les répertoires (-d) et les fichiers cachés (-a) :
   
[root@centos1 ~]# tree -ad /home
/home
├── nfs
└── tim
    └── .ssh
   
Lister en excluant :
   
tree -I node_modules
   
Exclure plusieurs (attention, ne pas mettre d'espaces avant et après les "|") :
   
tree -I 'node_modules|cache|test_*'
