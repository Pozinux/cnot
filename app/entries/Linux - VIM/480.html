Les commandes peuvent être précédées d'un facteur multiplicateur de leur action.
 
Par exemple :
 
- 7dd supprimera la ligne courante et les 6 lignes suivantes
- 3dl supprimera les 3 le caractère sous le curseur et les 3 suivants
 
dl ----- (delete letter) ----- suppression du caractère sous le curseur ----- ou la touche suppr du clavier ou "x"
dd ----- (delete) ----- suppression de la ligne courante
dw ----- (delete word) ----- delete current word
d0 ----- delete jusqu'au début de ligne
d$ ----- delete jusqu'à la fin de la ligne
 
u ------ (undo) ----- annule la dernière commande ---- comme crtl + z
ctrl + r --- (redo) --- comme ctrl + y
 
Supprimer toutes les lignes :
 
:1,$d
 
Supprimer les lignes 2 à 10  :
 
:2,10d
 
Supprimer les 10 premiers caractères de toutes les lignes : Regex
 
% : toutes les lignes
s/ / /g : substituer toutes les occurences trouvées sur la ligne
^.\{10\} : commençant par n'importe quel caractère répété 10 fois
 
:%s/^.\{10\}//g
 
Suppression des lignes commençant par un # :
 
:g/^#/d
 
Supprimer les espaces d'un fichier :
 
:1,$s/ //g
 
Supprimer les tabulations d'un fichier :
 
:1,$s/[\t]//g
