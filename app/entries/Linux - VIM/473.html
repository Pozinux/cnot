/chaine    // vers le bas à partir du curseur
?chaine    // vers le haut à partir du curseur
 
puis "n" pour next et "N" pour précédent.
 
On peut utiliser des REGEX pour la recherche :  Expressions régulières REGEX
 
BONUS : Si on veut rechercher le mot qui est sous le curseur dans le reste du texte, juste taper "*". (ESC avant)
 
Pour que la recherche ne soit pas case sensitive dans VI :
 
:set ic
 
ic = ignore case
 
Pour quelle redevienne case sensitive :
 
:set noic
 
Si c'est juste pour une recherche :
 
:/MOTACHERCHER\c
 
 
Highlitings :
 
Pour que la recherch soit surlignée :
 
:set hls
 
pour le supprimer en cours :
 
:nohlsearch     ou nohls
 
pour le supprimer tout le temps :
 
:set nohlsearch
 
Note : la completion marche pour les commandes après : pour trouver une option
