yy ----- (yank) ----- copie la ligne courante dans le buffer
yw ----- copie jusqu'au mot suivant
y0 ----- copie jusqu'au début de ligne
y$ ----- copie jusqu'en fin de ligne
p ------ (paste or put) ----- colle après le curseur / si c'est une ligne copiée avec yy il collera en dessous
P ------ (paste or put) ----- colle avant le curseur / si c'est une ligne copiée avec yy il collera au dessus
 
Toutes les commandes peuvent être remplacées par un facteur multiplicateur :
 
3yy ------- copier 3 lignes à partir de la ligne courante
3p  ------- coller 4 fois les 3 lignes copiées avant
 
Register :
 
vim a 26 register (buffer) nommés de a à z où l'on peut copier du texte et le garder pour le coller plus tard.
Il faut utiliser "
 
Copier dans le register "f" les 2 lignes :
 
2"fyy
 
On peut ensuite se déplacer et les coller ailleurs :
 
"fp
 
on peut les réutiliser plus tard, la copie reste dans le register.
 
Par défaut car rien n'est précisé c'est unamed register qui est utilisé. Mais lorsque l'on utilise un register, le unamed register est aussi mis à jour.
