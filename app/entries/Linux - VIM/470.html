Note : On peut afficher en bas sur quelle ligne on se trouve en faisant CTRL+g
 
 
: set nu
 
ou
 
: set number
 
Pour enlever les numéros de ligne :
 
: set nonu
 
Si on veut que ça s'affiche dès l'ouverture :
 
Créer un fichier .vimrc dans le répertoire personnel du user et y inscrire :
 
set number
 
ou toute option que l'on voudra s'exécuter à l'ouverture de vim.
 
Pas besoin de recharger la session, le fichier est pris en compte tout de suite.
