:w -------- (write) ------------- sauvegarde le fichier en cours d'édition
:q -------- (quit) -------------- abandonne l'édition
:q! ------- (quit) -------------- abandonne l'édition sans sauvegarder le fichier
:wq ------- (write and quit) ---- sauvegarde le fichier en cours d'édition et abandonne l'édition (idem :x)
 
Enregistrer le fichier en cours d'édition dans un autre fichier sans quitter le fichier courant :
 
:w new_fic
 
Note : 
 
On peut sélectionner une partie du texte en utilisant ESC+v puis faire :w file1 pour sauvegarder cette partie dans un fichier file1 (sans sortir du fichier actuel).
