cl ----- Change Letter
cw ----- Change Word
cc ----- Change line
~  ----- Changer la casse (maintenir enfoncé pour changer toute une ligne) ou + espace pour changer juste celui sous le curseur
 
Ne pas oublier qu'ils peuvent être préfixé d'un nombre :
 
3cl ----- Remplacer les 3 lettres suivantes
4cl ----- Remplacer les 4 mots suivants
 
 
Remplacer une chaine dans un fichier
 
Modèle de chercher-remplacer :
 
ranges/pattern/string/flags
 
RANGES :
 
1,5 = ligne 1 à 5
4 = ligne 4
% = toutes les lignes
1,$ = de la première à la dernière (donc équivalent à %)
 
FLAGS :
 
g = toutes les occurences de la ligne sinon si pas g alors seulement la première occurence trouvée
i = case in-sensitive
c = Demander confirmation avant de remplacer
 
Exemple :
 
:1,5s/chaine1/chaine2/g
 
Note : CTRL+g pour voir sur quelle ligne on se trouve
