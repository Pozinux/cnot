Coller la fonction dans le code (avant son utilisation) :
        
function debug() {
    $trace = debug_backtrace();
    $rootPath = dirname(dirname(__FILE__));
    $file = str_replace($rootPath, '', $trace[0]['file']);
    $line = $trace[0]['line'];
    $var = $trace[0]['args'][0];
    $lineInfo = sprintf('<div><strong>%s</strong> (line <strong>%s</strong>)</div>', $file, $line);
    $debugInfo = sprintf('<pre>%s</pre>', print_r($var, true));
    print_r($lineInfo.$debugInfo);
}

Afficher les erreurs
        
Sur un de mes sites, lorsque je fais une erreur dans php (par exemple j'oublie un ";" ) pour par exemple le site raspisms, je ne voyais juste que ça :
        
        
Pour afficher les erreurs, modifier le fichier suivant :
        
vim /etc/php/7.0/apache2/php.ini
        
Modifier display_errors = Off à On
        
Reload service Apache2
        
Au refresh de la page j'ai bien des indications (sinon on les trouve aussi dans le error.log d'Apache -> /var/log/apache2
