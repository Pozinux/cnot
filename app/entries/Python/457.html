POUR UN PACKAGE DEJA INSTALLE
 
Simplement :
 
$ pip show specloud
 
Package: specloud
Version: 0.4.4Requires:
nose
figleaf
pinocchio
 
Mais encore mieux :
 
Installer pipdeptree (pip install pipdeptree) puis par exemple pour le package pandas :
 
PS C:\Windows\system32> pipdeptree -p pandas
pandas==0.24.2
  - numpy [required: >=1.12.0, installed: 1.16.2]
  - python-dateutil [required: >=2.5.0, installed: 2.8.0]
    - six [required: >=1.5, installed: 1.11.0]
  - pytz [required: >=2011k, installed: 2018.9]
 
Note : pipdeptree tout court pour voir tous les packages du système et leurs dépendances
 
On peut aussi dans le sens inverse :
 
PS C:\Windows\system32> pipdeptree -r -p six
six==1.11.0
  - protobuf==3.6.1 [requires: six>=1.9]
    - mysql-connector-python==8.0.13 [requires: protobuf>=3.0.0]
  - python-dateutil==2.8.0 [requires: six>=1.5]
    - pandas==0.24.2 [requires: python-dateutil>=2.5.0]

POUR UN PACKAGE NON INSTALLE
 
Si c'est un package whl, on peut simplement le desarchiver avec 7zip par exemple et dans le répertoire "XXXX-X.X.X.dist-info" le fichier METADATA où l'on peut retrouver les dépendances.
 
ou bien le mieux est de l'installer sur un system qui a internet pour voir quelles dépendances il recherche.
